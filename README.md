# ULTRAKILL-Batching-Static-Fixer
This tool helps solve the issue with batching static breaking the environment impact effects and footsteps.

Tool is found at Bobot/Optimisation/PreservedOriginalMeshForStaticMeshes.

It works by using the PreservedOriginalMesh script from base game which would normally be handled by the static optimiser.

there are limits in terms of its capabilities firstly it is not dynamic at all it only applies the PreservedOriginalMesh the scripts when you use it so it needs to be used again to include new/modified meshes
